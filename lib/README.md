# External Libraries

Place the following JARs in this folder and refresh the Eclipse project:

- `amqp-client.jar` (RabbitMQ Java client)
- `cliente_http.jar` (HTTP client provided by instructors)
- `STC-DAO.jar` (STC DAO library provided by instructors)
- `mysql-connector-java.jar` (MySQL JDBC driver)
- `junit-jupiter-api.jar`, `junit-jupiter-engine.jar`, `apiguardian-api.jar`, `opentest4j.jar` (JUnit 5)

The `.classpath` file already references these names.
